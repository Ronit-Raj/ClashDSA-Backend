Two Sum

You are given an array of integers nums and an integer target. Find two distinct indices i and j such that:

nums[i] + nums[j] = target

Return the indices in increasing order (smaller index first). It is guaranteed that exactly one valid pair exists.

Input Format

  • The first line contains an integer n - size of the array.
  • The second line contains n space-separated integers nums[i].
  • The third line contains an integer target.

Output Format

Print two space-separated integers: the indices of the two numbers whose sum equals target, in increasing order.

Constraints

  • 2 <= n <= 200000
  • -10^9 <= nums[i] <= 10^9
  • -10^9 <= target <= 10^9
  • Exactly one solution exists.

Example 1

Input

4
2 7 11 15
9

Output

0 1

Example 2

Input

3
3 2 4
6

Output

1 2

Example 3

Input

2
3 3
6

Output

0 1

Notes

  • You must use two different indices.
  • Any approach with overall O(n) or O(n log n) time is acceptable.
