Word Break II

You are given a string s and a dictionary of strings wordDict. Insert spaces into s to construct all possible sentences such that every word in the sentence is a valid entry in wordDict. Return all such sentences in any order.

The same dictionary word may be reused multiple times in a single sentence. Every character in s must be consumed — no characters may be skipped or left over.

Input Format

  • The first line contains the string s.
  • The second line contains a single integer n — the number of words in the dictionary.
  • Each of the next n lines contains one dictionary word.

Output Format

Print all valid sentences, one per line, in any order. Each sentence must consist of words separated by single spaces, with no leading or trailing spaces.

If no valid segmentation exists, print nothing (empty output).

Constraints

  • 1 <= |s| <= 20
  • 1 <= |word| <= 10 for each word in wordDict
  • 1 <= wordDict.length <= 1000
  • All characters in s and in each dictionary word are lowercase English letters.
  • All words in wordDict are distinct.
  • Time limit: 3 seconds

Example 1

Input

catsanddog
5
cat
cats
and
sand
dog

Output

cats and dog
cat sand dog

Example 2

Input

pineapplepenapple
5
apple
pen
applepen
pine
pineapple

Output

pine apple pen apple
pineapple pen apple
pine applepen apple

Notes

  • The output order does not matter — any permutation of valid sentences is accepted.
  • A word from wordDict may be used more than once when constructing a sentence, as long as it appears in the dictionary.
  • Recommended approach: use backtracking with memoization. For each starting position in s, try every dictionary word as a prefix and recurse on the remaining suffix. Cache results per suffix to avoid redundant work.
  • Because |s| <= 20, even an unoptimized backtracking solution will fit within the time limit, but memoization is strongly encouraged to handle worst-case inputs with many overlapping subproblems.
