Longest Common Subsequence

Given two strings s and t, find the length of their longest common subsequence (LCS).

A subsequence of a string is formed by deleting zero or more characters without changing the relative order of the remaining characters.

Input Format

  • The first line contains string s.
  • The second line contains string t.

Output Format

Print a single integer: the length of the longest common subsequence.

Constraints

  • 1 <= |s|, |t| <= 3000
  • Both strings contain lowercase English letters.

Example 1

Input

abcde
ace

Output

3

Example 2

Input

abc
abc

Output

3

Example 3

Input

abc
def

Output

0

Explanation

In Example 1, one longest common subsequence is ace, so the answer is 3.

Notes

  • A standard dynamic programming solution runs in O(|s| * |t|) time.
