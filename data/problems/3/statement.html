Valid Parentheses

Given a string s containing only the characters '(', ')', '{', '}', '[', and ']', determine whether the string is valid.

A string is considered valid if all of the following hold:

  • Every opening bracket has a corresponding closing bracket.
  • Every closing bracket closes the most recently opened bracket of the same type.
  • Brackets are closed in the correct order.

Input Format

  • A single line containing the string s, consisting only of bracket characters.

Output Format

Print YES if the string is valid, or NO otherwise.

Constraints

  • 1 <= |s| <= 10^4
  • s consists only of the characters '(', ')', '{', '}', '[', ']'.
  • Time limit: 1 second.

Example 1

Input

()[]{}

Output

YES

Example 2

Input

([{}])

Output

YES

Example 3

Input

(]

Output

NO

Example 4

Input

([)]

Output

NO

Notes

  • A stack-based approach is the standard solution: push every opening bracket onto the stack and pop when a matching closing bracket is encountered.
  • If at any point the stack is empty when a closing bracket is seen, or the top of the stack does not match, the string is invalid.
  • After processing all characters, the string is valid only if the stack is empty.
