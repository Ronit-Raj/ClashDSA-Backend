Binary Search

You are given a sorted array of n distinct integers nums and an integer target. Return the 0-based index of target in the array. If target does not exist in the array, return -1.

The array is guaranteed to be sorted in strictly ascending order and contains no duplicate values. Your solution must run within the time limit, so a linear scan is not acceptable for large inputs.

Input Format

  • The first line contains an integer n - the number of elements in the array.
  • The second line contains n space-separated integers representing the sorted array nums.
  • The third line contains a single integer target - the value to search for.

Output Format

Print a single integer: the 0-based index of target in nums, or -1 if it is not present.

Constraints

  • 1 <= n <= 10^5
  • -10^9 <= nums[i] <= 10^9
  • -10^9 <= target <= 10^9
  • All values in nums are distinct.
  • nums is sorted in strictly ascending order.
  • Time limit: 1 second.

Example 1

Input

6
-5 1 3 7 12 19
7

Output

3

Example 2

Input

5
2 4 6 8 10
5

Output

-1

Example 3

Input

1
42
42

Output

0

Notes

  • Indices are 0-based: the first element is at index 0.
  • A binary search approach runs in O(log n) time and is expected for this problem.
  • Since all elements are distinct and the array is sorted, there is at most one valid answer.
