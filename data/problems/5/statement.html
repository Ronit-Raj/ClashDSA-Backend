Reverse Linked List

You are given a singly linked list represented as a sequence of n integers. Your task is to reverse the list and print the node values in reversed order.

A singly linked list is a data structure where each node holds a value and a pointer to the next node. Reversing it means the last node becomes the first, and so on.

Input Format

  • The first line contains a single integer n - the number of nodes in the linked list.
  • The second line contains n space-separated integers representing the values of the nodes from head to tail.

Output Format

Print n space-separated integers representing the node values of the reversed linked list, from the new head to the new tail.

If the list is empty (n = 0), print an empty line.

Constraints

  • 0 <= n <= 10^4
  • -10^5 <= node value <= 10^5
  • Time limit: 1 second

Example 1

Input

5
1 2 3 4 5

Output

5 4 3 2 1

Example 2

Input

1
42

Output

42

Example 3

Input

4
-3 0 7 -1

Output

-1 7 0 -3

Notes

  • The reversal must be done on the linked list structure itself, not simply by printing the array backwards, though the observable output is the same.
  • A recursive approach traverses to the end of the list and links nodes in reverse on the way back up the call stack.
  • An iterative approach uses three pointers (prev, curr, next) to reverse each link in a single pass, achieving O(n) time and O(1) extra space.
