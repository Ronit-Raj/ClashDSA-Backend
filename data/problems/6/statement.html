Longest Substring Without Repeating Characters

Given a string s, find the length of the longest substring that contains no repeating characters.

A substring is a contiguous sequence of characters within the string. Every character in the substring must appear exactly once.

Input Format

  • A single line containing the string s. The string may contain any printable ASCII characters including spaces.
  • If the string is empty, the input line will be blank.

Output Format

Print a single integer: the length of the longest substring of s that has no repeating characters.

Constraints

  • 0 <= |s| <= 10^5
  • s consists of printable ASCII characters.
  • Time limit: 2 seconds.

Example 1

Input

abcabcbb

Output

3

Example 2

Input

bbbbb

Output

1

Example 3

Input

pwwkew

Output

3

Explanation

In Example 1, the longest substring without repeating characters is abc, which has length 3. The substring starting at index 3 also yields abc, but no longer window is possible.

In Example 2, every character is b, so the best we can do is a window of size 1.

In Example 3, the answer is wke (length 3). Note that kew is also of length 3; either would be a valid witness, but only the length is required.

Notes

  • A sliding window approach with a hash set or a frequency array over ASCII codes solves this in O(|s|) time and O(1) space (bounded by the alphabet size).
  • Maintain two pointers left and right that define the current window. Advance right one character at a time; when a duplicate is detected, advance left until the duplicate is removed from the window.
  • If |s| = 0, the answer is 0.
